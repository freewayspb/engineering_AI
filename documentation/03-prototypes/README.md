## Как читать и проверять
1. Начните с этого `README.md` для обзора.
2. Перейдите в `prototype-1-machine-readable-extraction.md` для сценариев извлечения данных из машиночитаемых источников (OCR, штампы, сметы).
3. Откройте `prototype-2-data-normalization-templates.md` для нормализации данных и работы с шаблонами подразделений.
4. Просмотрите `prototype-3-knowledge-layer-queries.md` для интерактивного слоя знаний и обработки текстовых запросов.

Каждый прототип содержит:
- Demo Script (без метрик)
- Acceptance Checklist (без метрик)
- Sample data pack с относительными путями к `documentation/06-assets/input-documents/`

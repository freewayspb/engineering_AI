### 2.1. Сбор требований и анализ бизнес-потребностей

**Выполненные работы:**

**Документы:** `documentation/01-project-overview/project-vision.md`, `documentation/02-requirements/business-requirements.md`, `documentation/02-requirements/use-cases.md`

### 2.2. Исследование рынка AI/ML решений

**Выполненные работы:**
- Проведен анализ существующих AI/ML решений для локального развертывания
- Исследованы технологии OCR, NLP, обработки изображений и чертежей
- Проведен сравнительный анализ моделей по стоимости, эффективности, удобству интеграции
- Проанализированы требования безопасности и лицензирования
- Определены рекомендации по выбору технологического стека

**Результаты:**
- Отчет по анализу AI-моделей с обоснованными рекомендациями
- Определены технологии: Ollama для локального выполнения LLM, FastAPI для backend, Electron для клиента
- Рекомендации по выбору моделей для OCR и обработки документов

**Документы:** `documentation/08-tech-documentation/technical-specification.md`, `documentation/08-tech-documentation/solution-architecture.md`

### 2.3. Разработка и тестирование прототипов

**Выполненные работы:**
- Создан прототип P1: Извлечение данных из машиночитаемых источников (OCR, штампы, сметы)
- Создан прототип P2: Нормализация данных и работа с шаблонами подразделений
- Создан прототип P3: Интерактивный слой знаний и текстовые запросы
- Каждый прототип реализован как изолированный модуль для проверки ключевых гипотез
- Подготовлены Demo Script и Acceptance Checklist для каждого прототипа
- Сформированы Sample data packs с тестовыми документами

**Результаты:**
- Прототип 1: Модуль OCR для изображений и PDF (включая сметы, спецификации)
- Прототип 2: Алгоритмы группировки и нормализации, подготовка структурированных представлений
- Прототип 3: ML-модели для классификации и группировки документов
- Описания прототипов с гипотезами, выбранными технологиями и результатами тестирования

**Документы:** 
- `documentation/03-prototypes/prototype-1-machine-readable-extraction.md`

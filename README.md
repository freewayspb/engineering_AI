## Ключевые особенности

- **Локальное развертывание** без доступа к интернету
- **Интеллектуальная система обучения** на корпоративных документах
- **Текстовые запросы** на русском языке
- **Проектно-ориентированная работа** с данными
- **3 прототипа**: OCR, структурирование, классификация

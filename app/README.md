# BA AI GOST Client

Клиентское приложение для системы обработки документов BA_AI_GOST на основе Electron.js.

## Описание

Это десктопное приложение предоставляет пользовательский интерфейс для:
- Загрузки и обработки документов различных форматов (PDF, DWG, ARP, GSFX, XML, RTF, XLSX, DOCX)
- Извлечения структурированных данных из документов
- Экспорта результатов обработки
- Управления процессом обработки документов

## Структура проекта

```
app/
├── main.js          # Основной процесс Electron
├── preload.js       # Скрипт предзагрузки для безопасного API
├── index.html       # Главная страница приложения
├── renderer.js      # Логика пользовательского интерфейса
├── package.json     # Конфигурация проекта и зависимости
├── assets/          # Ресурсы приложения (иконки, изображения)
└── README.md        # Документация проекта
```

## Установка и запуск

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn

### Установка зависимостей

```bash
cd app
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Запуск приложения

```bash
npm start
```

### Сборка для распространения

```bash
npm run build
```

## Функциональность

### Основные возможности

1. **Загрузка документов**
   - Поддержка drag & drop
   - Диалог выбора файлов
   - Поддержка множественного выбора

2. **Обработка документов**
   - Автоматическое определение типа файла
   - Извлечение текста и метаданных
   - Оценка качества извлечения

3. **Управление результатами**
   - Просмотр результатов обработки
   - Экспорт в различные форматы
   - Сохранение настроек

4. **Интерфейс**
   - Современный дизайн с градиентами
   - Адаптивная верстка
   - Индикаторы прогресса
   - Статусная панель

### Поддерживаемые форматы

- **PDF** - PDF документы
- **DWG** - Чертежи AutoCAD
- **ARP** - Архивные файлы
- **GSFX** - Графические файлы
- **XML** - XML документы
- **RTF** - RTF документы
- **XLSX** - Excel таблицы
- **DOCX** - Word документы

## Архитектура

### Основной процесс (main.js)
- Управление жизненным циклом приложения
- Создание и управление окнами
- Обработка системных событий
- IPC коммуникация

### Процесс рендеринга (renderer.js)
- Пользовательский интерфейс
- Обработка пользовательских действий
- Взаимодействие с основным процессом

### Preload скрипт (preload.js)
- Безопасный мост между процессами
- Предоставление API для рендеринга
- Изоляция контекста

## Безопасность

- Context Isolation включен
- Node Integration отключен
- Remote Module отключен
- Безопасная IPC коммуникация
